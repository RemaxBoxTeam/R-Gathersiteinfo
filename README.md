# R-Gathersiteinfo
Information gathering by entering a host or a website then get full off information of the host ;)

# Installation 
> git clone https://github.com/RemaxBoxTeam/R-Gathersiteinfo/

> cd R-Gathersiteinfo

> pip3 install requests pyfiglet colorama 

> python3 gather.py
